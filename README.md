# OEmbed

Author : Cyril Pereira

## Install

```bash
yarn add funkymed-oembed --save
```

or 

```bash
npm install funkymed-oembed 
```

## Build

```bash
yarn build
```

## Test

```bash
yarn start
```

## In React

```javascript
import {OEmbed}  from 'funkymed-oembed/dist/index'
// ...
componentDidMount() {
    new OEmbed();
}
```
